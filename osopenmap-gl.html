A vector map made with OS OpenMap Local (Ordnance Survey) and INSPIRE index polygons (Land Registry) rendered on the client using the Mapbox GL Javascript API.

Requires Firefox, Chrome or Safari with WebGL enabled.

Right-click to rotate. Click the compass icon top-right to reset north.
